# Diploma Engineering College Website - Cuttack

This is a **static website** for a **Diploma Engineering College in Cuttack**. The website consists of multiple HTML pages covering various aspects of the college, including academic information, facilities, and student resources.

## 📂 File Structure

Below is the list of HTML files included in this project:

- `404-error.html` - Custom 404 error page  
- `about.html` - About the college  
- `approvals.html` - Approval details  
- `Cafeteria.html` - Cafeteria information  
- `campus.html` - Campus details  
- `canteen.html` - Canteen services  
- `contact.html` - Contact page  
- `courses.html` - List of courses offered  
- `faculties.html` - Faculty members' information  
- `feedback.html` - Feedback form/page  
- `gallery.html` - College image gallery  
- `hostel.html` - Hostel facilities  
- `index.html` - Homepage  
- `IndustryInstitute.html` - Industry collaborations  
- `lessionnotes.html` - Lesson notes for students  
- `lessionplan.html` - Lesson plan details  
- `library.html` - Library resources  
- `mission.html` - College mission and vision  
- `mou.html` - Memorandum of Understanding details  
- `placementcell.html` - Placement cell information  
- `PlacementDetails.html` - Placement details of students  
- `PlacementNotices.html` - Placement-related announcements  
- `skilldevelopment.html` - Skill development programs  
- `sports.html` - Sports and extracurricular activities  
- `timetable.html` - Academic timetable  
- `transportation.html` - Transportation facilities  
- `vision.html` - College vision  

## 🔹 Project Information

- **Type**: Static Website  
- **Technology Used**: HTML, CSS (if any), JavaScript (if any)  
- **Purpose**: Provides information about the Diploma Engineering College in Cuttack  

## 📌 Usage

This website is fully static and does not require a backend. You can host it on **GitHub Pages, Netlify, Vercel, or any static hosting service**. To run it locally, open `index.html` in any web browser.
